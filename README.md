express-seed
============

seed project for node.js + express

# Installation

1\. Clone this repo
```
git clone https://github.com/kimerran/express-seed.git
```
2\. Navigate to express-seed directory and run **npm install** to install dependencies
```
npm install
```

3\. Run the app
```
node app
```

# Dependencies
1. [node.js][1]
